# 작업 내역

- 프로젝트 이름을 goldstar로 확정하고 패키지 구조를 구성했습니다.
- 스플래시 화면과 팩 선택 화면을 구현했습니다.
- BLF_ 11개 리소스팩 검사 및 누락 경고/생성 흐름을 추가했습니다.
- 팩 설명 툴팁을 추가했습니다.
- CustomEntity 더블클릭 시 새 엔티티 생성 화면으로 이동하도록 했습니다.
- 엔티티 생성 입력 폼을 구성했습니다.
  - 필수: 이름/모델/텍스처
  - 선택: 애니메이션 컨트롤러/애니메이션/아이콘
- 엔티티 이름 규칙(영문 소문자/숫자/언더바, 최대 20자) 검증을 추가했습니다.
- prefix 입력(기본값 blf) 및 identifier 적용을 추가했습니다.
- 생성 시 실제 파일 복사/생성 및 item_texture.json 갱신을 연결했습니다.
- 로고를 로딩 화면 및 창 아이콘에 적용했고, 화질 개선 로직을 넣었습니다.
- BLF_CustomEntity 메타데이터 기본 템플릿을 활용하도록 연결했습니다.
- UI 텍스트를 한국어/영어 다국어 전환으로 구성했습니다.
- CustomEntity 생성 시 테스트용 행동팩(BLF_CustomTest) 생성 옵션을 추가했습니다.
- 행동팩에 엔티티(체력 1, 고정형) 및 스폰 아이템 최소 템플릿을 생성하도록 했습니다.
- 치장(커스텀 아머) 테스트용 아이템 4종 샘플을 행동팩에 생성하도록 했습니다.
- README의 pip 설치 안내를 제거했습니다.
